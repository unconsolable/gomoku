# Gomoku
A gomoku player via Min-Max Search and Alpha-Beta Pruning, running on https://botzone.org.cn/

### 使用方法
在线评测：在 botzone 上创建 bot，将 main_single.cpp 上传。

人机对局：在首页创建游戏桌，一个玩家设为 Bot，另一个设为房主。

本地调试：注释掉首行 ```#define ONLINE_JUDGE ``` 再运行。
